# Goldbach Deviation Framework

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18214509.svg)](https://doi.org/10.5281/zenodo.18214509)

**An Analytical Framework for Goldbach Deviation Structure: Spectral Representation, Persistence, and Amplitude Hierarchy**

## Overview

This repository contains the paper, data, and code for analyzing the structural properties of Goldbach deviations δ(N) = (G(N) - HL(N))/HL(N), where G(N) is the Goldbach representation count and HL(N) the Hardy-Littlewood prediction.

### Key Results

1. **Long-range persistence**: Hurst exponent H ≈ 0.84, attributed to L-function zero structure
2. **Two-tier amplitude hierarchy**:
   - Global envelope: κ_global ≈ C₂ (twin prime constant ≈ 0.6602)
   - Local fine structure: κ_local = 1/24 ≈ 0.0417
3. **Spectral correspondence**: FFT peaks at γ ≈ 6.02, 8.04, 14.13 matching L-function zeros

## Usage

### 1. Compute Goldbach counts and deviations
```bash
python scripts/compute_goldbach.py --nmax 100000 --output data/
```

### 2. Hurst exponent analysis
```bash
python scripts/compute_hurst.py --input data/goldbach_deviations.csv
```

### 3. FFT spectral analysis
```bash
python scripts/fft_analysis.py --input data/goldbach_deviations.csv
```

### 4. Coefficient fitting (1/24 formula validation)
```bash
python scripts/coefficient_fitting.py --input data/goldbach_deviations.csv
```

## Mathematical Framework

### Hardy-Littlewood Prediction
```
G(N) ~ 2 * C₂ * S(N) * N / (ln N)²
```
where C₂ = ∏_{p>2} (1 - 1/(p-1)²) ≈ 0.6601618158 is the twin prime constant.

### Spectral Representation (Conditional on GRH)
```
δ(N) = -2/ln(N) * Σ_χ χ̄(N) * Σ_γ cos(γ ln N + φ)/(1/4 + γ²) + O(1/(ln N)²)
```

### The 1/24 Formula
```
c_p = (1/24) * L(1, χ_p) / (p - 2)
```
